# Green-Tower
🟢 Green Tower - An out-of-this-world tower building game with alien themes, restaurants, retail, hospitals, security systems, and space exploration. Build the ultimate 10-star alien tower!

## Running Locally

Open `src/green-tower.html` in a browser (served from a local HTTP server so the JSON data file loads correctly). Example using Python:

```bash
python3 -m http.server 8080
# then open http://localhost:8080/src/green-tower.html
```

## Running Tests

```bash
npm install
npm test
```
